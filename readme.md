# 🌱 Agrinho 2026 – Futuro Sustentável, Agricultura Familiar e MST

Projeto desenvolvido por **Izabela Reche Mazia de Oliveira** para o **Concurso Agrinho 2026 – Paraná**, abordando temas como agricultura familiar, reforma agrária, soberania alimentar, agroecologia e a história do Movimento dos Trabalhadores Rurais Sem Terra (MST).

O site apresenta uma experiência moderna e interativa, utilizando animações, linha do tempo histórica, estatísticas dinâmicas e recursos visuais que promovem a reflexão sobre a importância da agricultura familiar para a construção de um futuro sustentável.

## 📸 Sobre o Projeto

O projeto foi criado com o objetivo de conscientizar sobre:

- Agricultura Familiar
- Sustentabilidade
- Reforma Agrária
- Soberania Alimentar
- Agroecologia
- Educação do Campo
- História e atuação do MST

Através de uma interface moderna, acessível e responsiva.
